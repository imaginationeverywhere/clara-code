Title: 92 Updates in Two Releases? Here's What Actually Matters!
Fetched: 2026-03-25T20:03:56.791Z
Segments: 84

--- Plain text ---
12 new features, 52 bug fixes, and 28 improvements across two releases. Starting with Enterprise, the new bare flag strips cloud code down to just the API. No hooks, no plugins, no LSP, perfect for CI pipelines. Teams can now drop independent policy files into a manage settings directory. And a new sandbox setting exits with an error if the sandbox can't start instead of silently running unprotected. Two new hook events land in this release. CWD changed files when you switch directories and file change files when files update on disk. Perfect for tools like dire andv. Agents can now declare an initial prompt in their front matter to autosubmit their first turn. And key bindings for killing agents and toggling fast mode are now fully rebindable through your keybindings config. The channels flag now relays tool approval prompts directly to your phone. Previously, this had issues with team and enterprise orcs and would show a not available message after upgrades [music] both are fixed. Remote control sessions now generate a I titles within seconds of your first message and fixes for rename syncing, exit archiving, and a memory leak in long sessions. Voice mode gets six fixes this cycle. Silent retry failures, dropped websocket connections, a 1 to 8 second U, I freeze on startup, Linux audio errors, and a space key that got swallowed after hold to talk. Beyond voice transcript mode now supports search with slash pasted images insert number chips at your cursor and Ctrl XR E opens your external editor. MCP tool calls now collapse into a synced queried server line. Expand with control O or off add support for the client ID metadata document spec and the MCP config flag can no longer bypass manage server policies. Plugins load from disk cache on startup instead of refetching and duplicate MCP servers from or connectors are automatically suppressed. Bedrock cold start latency drops by overlapping the profile fetch with other boot work. Resuming large sessions uses less memory and starts faster. Print mode startup saves 600 milliseconds by skipping unauthenticated MCP server handshakes. The non-streaming fallback jumps from 21,000 to 64,000 tokens with a 5m minute timeout and web fetch uses less peak memory on large pages. Multiple concurrent sessions no longer require repeated reauthentication when one refreshes its oorthth token. On Mac, clawed code no longer hangs on exit and the caffeinate process now terminates properly. So your Mac can actually sleep. Background agents that became invisible after context compaction are fixed along with a session history loss bug in the SD key. In VS Code, Windows path inheritance for Git bash is restored. A 60-second backend timeout now shows a not responding spinner and session history reloads correctly. Scroll back no longer jumps when collapsed groups finish or when the model starts stinking. Interrupting a prompt now restores your input so you can edit and resubmit and status works mid response. Run clot update in your terminal to get everything. Over 75,000 developers are now building with clot code on GitHub. Subscribe for weekly updates on every new release.
