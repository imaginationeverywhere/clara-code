## Low Priority Finding

### LOW-01 — No `workflow_dispatch` Inputs Defined
The manual trigger accepts no parameters. This means:
- You can't specify a custom `IMAGE_TAG` on a hotfix deploy
- You can't override the target environment
- Every manual trigger deploys HEAD of the selected branch

This is acceptable for now but worth adding as the ECS deploy matures. A future improvement:
```yaml
workflow_dispatch:
  inputs:
    image_tag:
      description: 'ECR image tag to deploy (default: latest commit)'
      required: false
      default: ''
```
